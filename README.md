# Datos-Mexico/observatorio

Repositorio institucional del Observatorio Datos México. Contiene
el manifesto y los artefactos vivos que articulan la identidad,
principios de trabajo y compromisos públicos del observatorio
con la sociedad mexicana.

## Documentos principales

- [`MANIFESTO.md`](MANIFESTO.md): Manifesto institucional del
  observatorio. Identidad, modelo de crecimiento por ciclos
  temáticos, arquitectura del corpus de respuestas, principios
  de trabajo académico y compromisos permanentes con el lector.

## Sobre este repositorio

Este repositorio versiona los artefactos institucionales del
observatorio. Los cambios sustantivos al manifesto se registran
en el historial de Git y se atribuyen al equipo académico que los
aprueba.

El observatorio recibe críticas, sugerencias y revisiones a
estos artefactos a través de sus canales de comunicación o
mediante Pull Requests a este repositorio.
